# T Coded Plumbing Website

Professional static website for T Coded Plumbing Service.

## Project Structure

```text
Tcoded/
  index.html
  assets/
    css/
      styles.css
    js/
      main.js
```

## What Each File Does

`index.html` contains the page content and section layout.

`assets/css/styles.css` controls the visual design, responsive layout, colors, spacing, cards, and animations.

`assets/js/main.js` controls the mobile menu, scroll effects, counters, testimonial slider, QR code, contact form message, particles, and WC flush demo.

## How To Preview Locally

Open `index.html` directly in a browser, or run a static server from the project folder:

```bash
python -m http.server 8010
```

Then visit:

```text
http://127.0.0.1:8010
```

## How To Deploy

This is a static website, so it can be deployed on Netlify, Vercel, GitHub Pages, cPanel, or any normal web hosting.

Upload the full project folder contents, keeping this structure:

```text
index.html
assets/css/styles.css
assets/js/main.js
```

## Before Going Live

Update these items in `index.html`:

- Phone number
- Email address
- Business address or service area
- Social media links
- Website URL in the QR input default value
- Real customer reviews if available

For best results, add real job photos or before-and-after plumbing work images. Real photos will make the site feel more trustworthy than icons alone.
